How to add a new SQL script if you are editing KInspector.Modules project:

- Add the SQL script into this folder
- On properties tab, click 'Copy always' from the dropdown
- Implement a new module using IModule interface
- Rebuild the KInspector.Modules
- Run the application

How to add a new SQL script if you are just adding a new DLL with modules:

- Add the SQL script into this folder
- Add the dll with modules into the parent folder
- Rerun the application
